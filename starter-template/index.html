Email starter kit

If you're a designer please check out or list of constraints before designing for an email template

If you're a developer please read the readme file before starting

Coding email templates can be a lot of work as they need to work across multiple clients (List of clients here) and the markup we normally use for websites won't work in emails. Emails still requires the extensive use of tables.
